-->Install Vader sentiment analysis as:

	$ pip install vaderSentiment (use pip in case pip3 doesn't work for python 3.6.9)
	
-->Import into .py file as
	from vaderSentiment.vaderSentiment import SentimentIntensityAnalyzer


-->For converting emojis and emoticons to text
	-install fasttext as follows:
		$ pip install fasttext(use pip in case pip3 doesn't work for python 3.6.9)
